# gorent

Go Rent is an backend API for vehicle rent system built on Go Language

## Installation

1. Clone the repository

```bash
git clone https://github.com/chirzul/gorent.git
```

2. Install dependencies

```bash
go mod tidy
```

3. Environment Variable

### Database

```bash
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

4. Run the app

```bash
go run main.go
```

:fire: You're all set!
